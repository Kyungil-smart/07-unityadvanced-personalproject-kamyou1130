# BOSS SLAYER

## 프로젝트 설명

- 게임 제목 : BOSS SYLAER

- 난이도 : 최하

- 개발 환경 : Unity 6000.3.9f1

- 플랫폼 : Windows

- 프로젝트 기간 : 7일

## 게임 특징
  
- 3D 탑다운 뷰 게임으로 닌텐도 DS '젤다의 전설 몽환의 모래시계' 게임을 참조
  
- 보스 몬스터를 처치하면서 진행
  
- 각 보스 몬스터의 공격 패턴을 파악하고 파훼하여 처치

## 게임 플레이 방법

- W/A/S/D : 플레이어 이동
  
- 마우스 우클릭(유지) : 플레이어 오브젝트 공격 가능 상태
  
- 마우스 좌클릭 : 공격
  
- Ctrl : 폭탄 설치
  
- Shift : 대쉬
